Sets the height and width of the character in pixels. Data vector of strings is used to represent the pixel states.

Each char in the string sets the pixel on or off: space ' ' is off; anything else is on. Font pixels are stored top to bottom, left to right. So the top left hand corner pixel of the font character is stored at data[0][0]

Using this representation means that it's easy to "draw" the font characters direct into code
